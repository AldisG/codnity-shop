# React boilerplate

### It contains:

- [✔️] React + TypeScript
- [✔️] Redux Toolkit + with types via useAppSelector / useAppDispatch
- [✔️] Create Api
- [✔️] Route
- [✔️] React Testing Library and Jest
- [✔️] Webpack setup
- [✔️] Framer Motion
- [✔️] ESLint and Prettier setup (plugins still required)

* To run the project - npm start
* To run tests - npm run test
